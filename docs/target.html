Localization protocol

A description of the protocol of particle localization

ThunderSTORM (ImageJ plugin)

Input: a TIFF file containing multiple images of fluorescent microscopy

Output: particle locations with width, height, and background level of fitted point-spread function

  1. Raw image I is filtered with B-spline wavelet filters twice to produce F1 and F2
  2. 8-connected local maxima xi are located in F2 with I(xi) > 1.5×STD(F1)
  3. For each maximum xi, a square region of size (2l+1)2 is used to fit a integrated-Gaussian point-spread function for the most-likely values of the parameters: center, xc and yc, width σ, height h, and background b
